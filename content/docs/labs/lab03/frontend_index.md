# Lab03 前端基础

## 实验目的

- 了解 Vue 框架的基本使用
- 熟悉 Vue CLI 的基础命令
- 掌握 Vue-Router、Vuex 与 Vue 的集成
- 掌握 ElementUI 的基础使用
- 掌握使用 Axios 发送网络请求的方法
- 熟悉 Vue 单页面组件的使用
